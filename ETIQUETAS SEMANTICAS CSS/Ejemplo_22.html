Modal con CSS Puro

Este ejemplo muestra cómo crear un modal funcional utilizando solo HTML y CSS, sin necesidad de JavaScript. El modal se activa mediante la pseudo-clase :checked de un input checkbox.

¡Este es un modal!

Puedes crear modales completamente funcionales utilizando solo CSS. Esto se logra mediante el uso del selector :checked para alternar la visibilidad del modal.

El modal incluye una animación suave al aparecer y desaparecer, y se cierra haciendo clic en el botón de cierre o en cualquier parte fuera del contenido del modal.
