Preguntas Frecuentes

¿Qué es HTML5?

HTML5 es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML. HTML5 especifica dos variantes de sintaxis para HTML: un «clásico» HTML (text/html), la variante conocida como HTML5 y una variante XHTML conocida como sintaxis XHTML5 que deberá ser servida como XML (XHTML).

Esta es la primera vez que HTML y XHTML se han desarrollado en paralelo. El estándar incluye novedades significativas como funciones para incrustar audio, video, gráficos, almacenamiento de datos en el lado del cliente y contenido interactivo sin necesidad de plugins.

¿Cuáles son las principales características de CSS3?

CSS3 introduce numerosas características nuevas que permiten crear diseños más sofisticados. Algunas de las más destacadas incluyen:

  • Bordes redondeados (border-radius)
  • Sombras para cajas y texto (box-shadow, text-shadow)
  • Gradientes de color
  • Animaciones y transiciones
  • Transformaciones 2D y 3D
  • Consultas de medios para diseño responsivo
  • Flexbox y Grid para layouts avanzados
¿Qué ventajas ofrecen las etiquetas semánticas de HTML5?

Las etiquetas semánticas de HTML5 como <header>, <footer>, <article>, <section>, <nav>, <aside> y <main> ofrecen varias ventajas:

  1. Mejor estructura y organización del código
  2. Mayor accesibilidad para lectores de pantalla
  3. Mejor posicionamiento SEO al ayudar a los motores de búsqueda a entender la estructura del contenido
  4. Código más legible y mantenible
  5. Facilita el estilo con CSS al tener elementos con significado claro
¿Cómo funciona el modelo de caja en CSS?

El modelo de caja en CSS es fundamental para entender cómo se representan los elementos en una página web. Cada elemento es considerado como una caja rectangular que consta de:

  • Contenido (content): El área donde aparece el contenido (texto, imágenes, etc.)
  • Relleno (padding): Espacio transparente alrededor del contenido
  • Borde (border): Línea que rodea el padding y el contenido
  • Margen (margin): Espacio transparente fuera del borde que separa el elemento de otros

El tamaño total de un elemento se calcula sumando el ancho/alto del contenido + padding + borde. Los márgenes no se cuentan para el tamaño total pero afectan el posicionamiento respecto a otros elementos.
