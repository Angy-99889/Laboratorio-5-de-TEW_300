Menú Desplegable con CSS Puro

Este ejemplo muestra cómo crear un menú desplegable multinivel utilizando únicamente HTML y CSS, sin necesidad de JavaScript. Se utilizan selectores de pseudoclase como :hover para controlar la visibilidad de los submenús.

Las transiciones CSS suavizan la aparición y desaparición de los submenús, creando una experiencia de usuario más agradable.
