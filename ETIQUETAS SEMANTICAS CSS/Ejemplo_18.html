Menú Hamburguesa con CSS Puro

Este ejemplo muestra cómo crear un menú hamburguesa completamente funcional utilizando solo HTML y CSS, sin necesidad de JavaScript.

El menú se activa al hacer clic en el icono de hamburguesa y se desliza desde el lado derecho de la pantalla.

En dispositivos más grandes (ancho mínimo de 768px), el menú se muestra normalmente en la barra de navegación.
